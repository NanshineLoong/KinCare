# Contributing

Thank you for your interest in contributing to KinCare!

We welcome all contributions — from bug fixes to new features.

## Getting Started

1. **Fork** the repository

2. **Clone** your fork

```bash
git clone https://github.com/<your-username>/kincare.git
cd kincare
````

3. **Create a branch**

```bash
git checkout -b feature/your-feature-name
```

> Use `feature/xxx` or `fix/xxx` for branch names.

4. **Make your changes**

* Follow the existing code style
* Ensure your changes work as expected

5. **Commit your changes**

```bash
git add .
git commit -m "feat: add your feature"
```

> We recommend using [Conventional Commits](https://www.conventionalcommits.org/).

6. **Push to your fork**

```bash
git push origin feature/your-feature-name
```

7. **Open a Pull Request**

* Target branch: `main`
* Clearly describe what your PR does
* Link related issues if applicable

## Not Sure Where to Start?

Feel free to open an issue or discussion — we're happy to help!

## Code of Conduct

Please be respectful and constructive in all interactions.
